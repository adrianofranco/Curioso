h1. Curioso

p. Curioso, Curious in english, is a Class for make scrape in html pages

h2. Using

h3. Simple Class
	
@$curioso = new Curioso();@

@$curioso->scrape("http://www.google.com", "/html/body")@

h3. Pass format

	@$curioso = new Curioso();@
	
	@$curioso->format="object"; //can be (object | xml | array | json)@
	
	@$curioso->scrape("http://www.google.com", "/html/body")@
